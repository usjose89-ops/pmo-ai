ESTÁNDARES DE ALMACENAMIENTO - CONSTRUCTIA

Estructura de Carpetas para Proyectos:

1. 1_BASES
   - Administrativas: Bases administrativas especiales y generales.
   - Tecnicas: Especificaciones técnicas, planos, mecánica de suelos.
   - Generales: Normativas, declaraciones de impacto, etc.

2. 2_PRESUPUESTO (Área de Estudios)
   - Cubicaciones, Cotizaciones, Análisis de Precios Unitarios (APU).
   - Presupuesto Oficial y Presupuesto Meta.

3. 3_INFORMACION
   - RDI (Requerimientos de Información).
   - Comunicaciones con el Cliente / ITO.
   - Libros de Obra Digitales.

4. 4_OTROS
   - Fotos, Videos, Drone.
   - Boletas de Garantía.
   - Contratos de Subcontratos.
